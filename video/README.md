# Видео-приветствия

Эта папка содержит приветственные видео для каждой категории аудиоконтента.

## Структура

```
video/
├── mindfulness/intro.mp4    # Приветствие для медитаций осознанности
├── thoughts/intro.mp4        # Приветствие для "Мыслей вслух"
└── podcast/intro.mp4         # Приветствие для «Психопогромизма»
```

## Требования к видеофайлам

### Технические характеристики
- **Формат**: MP4 (H.264)
- **Разрешение**: 1920x1080 (Full HD) или 1280x720 (HD)
- **Битрейт видео**: 2-5 Mbps
- **Битрейт аудио**: 128-192 kbps
- **Кодек видео**: H.264
- **Кодек аудио**: AAC
- **Название файла**: всегда `intro.mp4`

### Содержание
- **Длительность**: рекомендуется 1-3 минуты
- **Контент**: краткое приветствие и описание категории
- **Тон**: спокойный, располагающий (соответствует тематике майндфулнесс)
- **Фон**: желательно нейтральный или природный

## Как добавить видео

1. Подготовьте видеофайл согласно требованиям выше
2. Поместите его в соответствующую папку категории
3. Убедитесь, что файл называется `intro.mp4`
4. Проверьте, что путь в `src/config/site.ts` правильный:

```typescript
categories: {
  mindfulness: {
    videoIntro: '/video/mindfulness/intro.mp4',
    // ...
  },
  // ...
}
```

## Конвертация видео (если нужно)

Используйте FFmpeg для конвертации видео в нужный формат:

```bash
# Конвертация в MP4 H.264 с оптимальными настройками
ffmpeg -i input.mov -c:v libx264 -preset slow -crf 22 -c:a aac -b:a 128k -movflags +faststart intro.mp4
```

Параметры:
- `-c:v libx264`: кодек H.264
- `-preset slow`: лучшее качество при сжатии
- `-crf 22`: constant rate factor (18-28, меньше = лучше качество)
- `-c:a aac`: аудио кодек AAC
- `-b:a 128k`: битрейт аудио
- `-movflags +faststart`: оптимизация для веб (progressive download)
